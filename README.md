# База знаний преподавателей

Этот репозиторий — внутренняя база знаний для преподавателей курсов Код Будущего.

Цель базы:
- сократить количество повторяющихся вопросов;
- фиксировать уже найденные решения типовых проблем;
- дать преподавателям возможность самостоятельно находить ответы;
- аккумулировать практический опыт работы в учебных аудиториях.

## Что здесь будет
В базе знаний планируется хранить:
- ответы на частые вопросы преподавателей;
- инструкции (установка ПО, настройка среды, типовые ошибки);
- решения проблем, возникающих в аудиториях и компьютерных классах;
- ссылки на обсуждения и уточнения.

Основной формат — **вопрос → причина → решение → примечания**.

## Как пользоваться
- Для чтения решений используется **Wiki** этого репозитория.
- Если нужного ответа нет — создаётся **Issue** с описанием проблемы.
- После появления решения оно фиксируется в Wiki, чтобы вопрос не повторялся.

Принцип работы:
> Сначала поиск в базе — потом вопрос.

База предназначена для внутреннего использования и не является учебным пособием для студентов.

## Статус
База знаний находится в стадии формирования.  
Структура и наполнение будут дополняться по мере накопления вопросов и решений.
